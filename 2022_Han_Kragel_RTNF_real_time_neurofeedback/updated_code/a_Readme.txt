Readme

This experiment was created by Philip Kragel in 2019.
It is being used to run the CANlab RTNF experiment 1,
on pain regulation using neurofeedback.
The study reduces NPS activity and tests effects on pain and emotion.

It uses the LabJack Matlab object-oriented software
https://labjack.com/support/software

and FieldTrip
http://www.fieldtriptoolbox.org/
